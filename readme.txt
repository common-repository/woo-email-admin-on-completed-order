Woocommerce Email Admin On Completed Order


== Description ==


When order is completed confirmation email will be sent out to admin as well .




== Installation ==



1. Upload `woocommerce_email_admin_on_completed_order` folder to your `/wp-content/plugins/` directory.

2. Activate the plugin from Admin > Plugins menu.
